## Invisible features / notes
- **G1 open-in-external-editor**: surfaced in three in-scope places — `Navbar` (`OpenInIdeButton` for single-repo projects), `DiffCard` (per-file `attemptsApi.openEditor`), and `ConflictBanner` (`onOpenEditor` button). `EditorAvailabilityIndicator` reports whether the configured editor exists. If G1 is deleted, `EditorAvailabilityIndicator` + the Navbar/DiffCard open buttons all become removable.
- **VS Code webview bridge**: none in this scope (ide/ and vscode/ excluded).
- **Quality Gate System A / AuditPlan System B / planning-draft materialize flow**: none present in this scope.
- **Legacy vs New design split**: `LegacyDesignScope` (this scope) vs `NewDesignScope` (out of scope). Legacy now only backs `/commands` and two redirects; the `panels/` + root `DiffCard` triad look like leftovers from the legacy task-detail UI that the ui-new ChangesPanel replaced.
