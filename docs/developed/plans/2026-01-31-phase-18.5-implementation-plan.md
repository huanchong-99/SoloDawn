## 3. 风险与缓解措施

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| `shared/types.ts` 是生成文件 | P0-4 需要后端配合 | 先在前端定义扩展类型，后续同步后端 |
| WS 协议升级影响现有功能 | 可能导致终端通信中断 | 保持向后兼容，新旧协议并存过渡 |
| dnd-kit 与现有样式冲突 | 拖拽视觉异常 | 隔离拖拽样式，使用 CSS 变量 |
| 模型 API 涉及密钥安全 | 密钥泄露风险 | 前端不持久化明文密钥，使用后端代理 |
| 终端数量扩展增加资源消耗 | 性能问题 | 后端限流校验，前端提示资源占用 |
